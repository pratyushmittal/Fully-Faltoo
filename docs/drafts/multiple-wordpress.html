Posted on Wed 15 April 2015

I recently migrated all my Wordpress installations spread across different web-hosts to a single DigitalOcean server. This is a brief step-by-step checklist for the reference purpose.

  1. Setup a LAMP stack: Follow this guide for setting up mysql and apache configurations. Mirror link.

  2. Take a backup of uploads on source server:

# cd into wp-content folder or source server zip -ruq uploads.zip uploads/ mv uploads.zip uploads/

  1. Downloading media files:

    cd into wp-content folder on destination server

    wget http://website.com/wp-content/uploads/uploads.zip rm uploads unzip uploads.zip

  2. Enable mod-rewrite

  3. Update hosts
  4. Import export
  5. change dns
  6. plugins, disqus
  7. Settings -> Permalinks
  8. Appearance
